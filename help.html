LocalWriterAI

Características

Extender Selección

Esta función te permite expandir tu selección actual de texto con contenido generado por IA que coincide con el contexto y estilo de tu documento.

Editar Selección

Selecciona texto y usa esta opción para que la IA te ayude a editar y mejorar el contenido seleccionado mientras mantiene tu mensaje original.

Configuración

Configura tus preferencias de LocalWriterAI, incluyendo la configuración de la API y las opciones de idioma.

Consejos de Uso

  • Selecciona el texto antes de usar las funciones de Extender o Editar
  • Asegúrate de que la configuración de la API esté correctamente configurada
  • La extensión funciona mejor con selecciones claras y específicas

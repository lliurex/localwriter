Ayuda del Plugin LocalWriterAI

Introducción

El plugin LocalWriterAI integra funcionalidades de inteligencia artificial generativa directamente en LibreOffice Writer, facilitando la generación y edición de textos. Antes de utilizarlo, es necesario realizar configuraciones previas en tu equipo.

Pasos Previos para Utilizar LocalWriterAI

1. Instalar Ollama

Ollama es una plataforma versátil que permite ejecutar modelos de lenguaje de gran tamaño (LLMs) localmente en tu equipo. Para instalarlo en LliureX, sigue estos pasos:

  • Abre Zero Center desde el menú de aplicaciones de LliureX.
  • Busca Ollama y sigue las indicaciones para instalarlo.
  • Verifica la instalación ejecutando ollama --version en la terminal (konsole).

2. Descargar un Modelo LLM

Una vez instalado Ollama, es necesario descargar un modelo de lenguaje, como por ejemplo Qwen2.5. Esto se realiza desde la terminal:

  • Abre konsole y escribe: ollama pull qwen2.5.
  • Alternativamente, puedes explorar otros modelos disponibles en la librería oficial de Ollama.
  • Verifica que el servidor esté activo ingresando http://127.0.0.1:11434 en tu navegador.

3. Configurar LocalWriterAI en LibreOffice Writer

Tras instalar el plugin desde Zero Center, sigue estos pasos para configurarlo:

  • Asegúrate de que el servidor de Ollama esté activo.
  • Abre LibreOffice Writer y accede al menú LocalWriterAI > Configuración.
  • Introduce la URL del servidor: http://127.0.0.1:11434, selecciona el modelo (qwen2.5) y ajusta los límites de tokens según tus necesidades.

Ejemplos de Uso de LocalWriterAI

Extender Selección

Selecciona un fragmento de texto en tu documento y utiliza la opción para generar contenido adicional relacionado con el texto seleccionado.

Ejemplo: «La Revolución Industrial fue un periodo de grandes avances tecnológicos y sociales que...»

Editar Selección

Selecciona un texto y proporciona instrucciones para modificarlo, como cambiar el tono, traducir o resumir.

Ejemplo: Cambiar el tono de una carta informal a uno formal.

Conclusión

Integrar herramientas como Ollama y LocalWriterAI en LliureX mejora significativamente tu productividad, permitiéndote aprovechar las tecnologías de IA de forma eficiente y privada. 🚀
